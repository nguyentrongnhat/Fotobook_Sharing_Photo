* To run this project (Fotobook) you must:
1. Install ruby version 2.7.0 + devkit(IMPORTANT => version: 2.7.0)
2. Install rails 6
3. Install yarn
4. Install Node js
5. Install Sqlite3
6. use command line cd into project
7. run this command: bundle install
8. run this command: rails server /*(wait until you see "Use Ctrl-C to stop")*/
9. Open browser and enter: localhost:3000
